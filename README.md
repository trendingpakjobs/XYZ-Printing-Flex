# XYZ Printing Website

Professional, modern, and fully responsive website for a printing business. Designed for showcasing services like flex printing, banners, visiting cards, and stickers. Includes animated service cards, portfolio section with images, and WhatsApp integration for easy client orders.

## Features

- Fully responsive (mobile & desktop)  
- Clean, premium, and professional UI  
- Animated service cards for modern look  
- Portfolio section with images  
- WhatsApp button for direct orders  
- Easy to customize for any printing business  

## Technologies Used

- HTML5  
- CSS3 (Grid, Flexbox, Animations)  
- Remix Icons
